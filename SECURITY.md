# Security Policy

## Supported Versions

We support security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately by opening a security advisory or contacting the maintainers directly. Do NOT disclose the vulnerability publicly in github issues until we have had an opportunity to review and address it.
