# Contributing to ExamAI CLI

Thank you for choosing to contribute to ExamAI CLI! We welcome community contributions to help improve this terminal assistant for engineering students.

## Code of Conduct
By participating in this project, you agree to abide by our Code of Conduct (detailed in [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)).

## How Can I Contribute?
- **Reporting Bugs**: Open an issue describing the bug, steps to reproduce, and environment details.
- **Suggesting Enhancements**: Suggest new features or modes (e.g. new subjects or formats).
- **Submitting Pull Requests**:
  1. Fork the repository and create your branch from `main`.
  2. Install dependencies: `pip install -e .` and `pip install pytest`.
  3. Make your changes, ensuring adherence to PEP 8 coding conventions and type hint requirements.
  4. Write unit tests for new logic.
  5. Run the test suite: `pytest` and verify that all tests pass.
  6. Submit the PR for review!
